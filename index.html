Recent work👔

The Girl Code

The Girl Code is a Non Profit Organization with offices in New Delhi, Hyderabad and Singapore. It aims to introduce young girls to programming by hosting workshops at local schools.

View project

Sugar Labs

Sugar is an activity-focused, open-source learning platform for children. It is a member project of the Software Freedom Conservancy and serves as a support base and gathering place for the community of educators and software developers interested in EdTech.

Project Element

Project Element was an initiative to help students find their passion through a series of interactive workshops which benefitted over 3,000.
